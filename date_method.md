# Date method

```js
var today = new Date(); 
```

**Example: **

```js
var today = new Date(2000,5,19);
```

###.getMonth();

```js
today.getMonth(); // 5
```

###.getFullYear();

```js
today.getFullYear(); // 2000
```

###.getDate();

```js
today.getDate(); // 19
```

###.getDay();

```js
today.getDay(); // 1 (Monday)
```
--> 0 = Sunday

###.getHours();

```js
today.getHours(); 
```
--> 0 - 23

###.getTime();

```js
today.getTime();
```

--> Milisecond since 1/1/1970
