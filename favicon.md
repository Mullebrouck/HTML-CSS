# Favorite Icon

= image in address bar

**Size**
- 16x16 pixels
- 32x32 pixels

**Extension**

save as a "favicon.ico" 

** Attributes **

- rel= value is icon
- href = name of image
- type = MIME type of image (image/x-icon)


```html
<link 
    rel="icon" 
    href="favicon.ico"
    type="image/x-icon"
>
Also add this <link> for IE
<link
    rel="shortcut icon" 
    href="favicon.ico"
    type="image/x-icon"
>

```
