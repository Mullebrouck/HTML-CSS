# Image hyperlinks

to create an image hyperlink: 

```html
<a href="index.html"> 
    <img 
        src="home.gif" 
        height="19"
        width="85"
        alt="Home"
    >
</a>

/*Hyperlink will display by adding blue line, if you don't want the blue line add `border` property*/

<a href="index.html"> 
    <img 
        src="home.gif" 
        height="19"
        width="85"
        alt="Home"
        border="0"
    >
</a>

```
