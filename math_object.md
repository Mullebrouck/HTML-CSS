# Math object

### Math.round(); 

```js
var x = 99.9; 
var y = Math.round(x); //100
```

### Math.max(); 

```js
var a = 100, b = 50, c = 88;   
var biggest = Math.max(a,b,c); //100
```

### Math.min(); 

```js
var a = 100, b = 50, c = 88;   
var smallest = Math.min(a,b,c); //50
```

### Math.PI; 

```js
Math.PI; //3.141592653589793
```
