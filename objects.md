# Objects

Declaring and defining (No classes in JavaScript, only objects)



## Notation:

**Literal Notation**

```js 
var myObject = {
  key: value, 
  key: value, 
  key: value
};
```
**Constructor Notation**

```js 
// Tells you we are creating a new object
var myObj = new Object();

myObj.property = some value;
//or
myObj["property"] = some value;
```
